# Proyecto Final Desarrollo Web

## Descripcion del proyecto:

> Carieva es un estudio de decoración de interiores y diseño interior, que se dedica a la venta de productos y asesoramiento personalizado para crear el perfecto hogar del cliente.

## Tecnicas utilizadas:

> Para el prototipado de la pagina web del estudio se utilizo HTML y CSS. Ademas, se adaptaron alguns Bootstrap para complementar el SASS, y se utilizo la plataforma de Github como repositorio local.

## Avances realizados:

>** Commit Desafios 1 ** : Se inicio la primer base de la pagina, con solamente la utilizacion de HTML para su estructura en relacion a los wireframes creados.

>** Commit Desafios 2 ** : Se inicio la primera aplicacion de CSS a la pagina.

>** Commit Desafios Primera entrega ** : Se realizo los HTML y se aplico el CSS a todas las paginas de la plataforma.

>** Commit Desafios 3 y Desafio complementario 1 ** : Se aplicaron displays grids a toda la pagina, teniendo en cuenta que la plataforma quede completamente responsive.

>** Commit Bootstrap ** : Se exploro la utilizacion de git y la utilizacion de diferentes ramas para la exploracion de las animaciones y bootstraps.

>** Commit Desafios 4 y complementario ** : Se aplicaron animaciones en toda la pagina, utilizando en algunos casos adaptaciones de referentes de bootstraps.

>** Commit Segunta entrega ** :  Se ayustaron detalles de animaciones y se verifico que la plataforma sea completamente responsive.

>** Commit Desafios 5** : Se inicio las primeras aplicaciones de SASS al proyecto.

>** Commit Desafios 6 ** : Se aplico con mayor profundidad SASS a todo el proyecto. Ademas se realizo una optimizacion del proyecto aplicando los conceptos SEO.

>** Commit Desafios Tercera entrega ** : Se ajustaron todos los detalles de SASS en toda la platafomra, siempre manteniendo la platafomra completamente responsive.

>** Commit Desafios Avance entrega ** : Se hicieron cambios detallistas para la mejora de la presentacion del proyecto. Ademas de incorportar todas las nuevas imagenes representativas del proyecto.

>** Commit Desafios Entrega Proyecto Final ** : Se hicieron los ulitmos ajustes de la pagina web para la finalizacion del proyecto.
